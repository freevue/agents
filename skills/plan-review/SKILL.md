# Plan Review

`implementation_plan.md`나 `task.md`에 대한 사용자의 리뷰를 파악하여 개선된 내용으로 문서를 업데이트합니다.

## 핵심원칙

1. **규칙 준수**: [규칙](/Users/juno/.gemini/PLAN_CONVENTIONS.md)을 다시 한번 상기하여, 절대적으로 template을 준수합니다.
2. **이전 문서 기억**: 리뷰가 중복되며 내용이 사라지는 것을 주의합니다. 해당 문제를 방지하기 위해 진행중인 리뷰와 이전 문서의 내용을 동시에 파악합니다.
3. **사용자의 요청 사항 비교**: 개선된 문서가 사용자의 요청에 맞추어 부합한 내용인지 확인합니다. 사용자의 요청을 다시 불러와 문서와 비요하여 잘못된 부분이 없는지 확인합니다.
