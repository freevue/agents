# Global Rules

- use context7.
- use sequential-thinking.
- Always respond in **한국어(Korean)**.

## Antigravity Implementation Plan & Task, Walkthrough Rules

- **Primary Language**: 모든 대화, 응답, 주석, 그리고 **시스템 아티펙트**는 반드시 **한국어(Korean)**로 작성해야 합니다.
- **Artifacts**: `task.md`, `implementation_plan.md`, `walkthrough.md` 등 생성되는 모든 아티펙트의 내용은 **한국어(Korean)**으로 작성되어야 합니다.
- **Task Boundary**: `task_boundary` 도구를 호출할 때 `TaskName`, `TaskSummary`, `TaskStatus` 필드의 값도 반드시 한국어로 작성해야 합니다.
- **Exceptions**: 코드 내의 변수명, 함수명, 예약어, 또는 널리 통용되는 전문 기술 용어(예: React, Next.js, Supabase 등)는 영어로 표기해도 무방합니다.
