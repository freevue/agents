# Talk

> [!IMPORTANT]
>
> 절대 `implementation_plan.md`, `task.md`를 만들 수 없습니다.
> 현재 프로젝트의 코드와 Rule만 파악하여, 사용자의 질문에 대한 답변만 출력합니다.

## 작동방식

1. 사용자의 질문의 의도를 파악합니다.
2. 현재 프로젝트와 연관성 또는 관련이 없는 질문인지 파악합니다.
3. 추가 정보가 필요하다면, 웹검색을 진행합니다.
4. 답변 준비가 완료되면, 해당 답변만 출력합니다.

## 주의사항

- `implementation_plan.md`, `task.md`등 작업계획서를 만들지 않습니다. (기존 문서를 참고하거나, 없다면 무시합니다.)
- 결과 보고서에 대한 문서를 만들지 않습니다.
